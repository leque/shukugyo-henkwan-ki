# shukugyo-henkwan-ki
音読みする熟語を入力するとそれを日本漢音の字音仮名遣いに変換します。字音の選択は手動です。

## 元データ
字音のデータは[日本漢音廣韻全字表.csv](https://github.com/leque/textfile/blob/master/kan-on-all/%E6%97%A5%E6%9C%AC%E6%BC%A2%E9%9F%B3%E5%BB%A3%E9%9F%BB%E5%85%A8%E5%AD%97%E8%A1%A8.csv)から取っています。

## 使い方
[熟語変換器](https://leque.github.io/shukugyo-henkwan-ki/index.html)にアクセス、熟語を入れて「Henkwan」を押そう
